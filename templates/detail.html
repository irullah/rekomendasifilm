Persamaan:
\(Top(l_u,N)\) \(=\) \(\{l_u(1), l_u(2), ..., l_u(N)\}\)
\(s_u\) \(=\) \( |Top(l_u,N) \, \cap \, Test_u|\)
\(Precision(Test_u,l_u,N)\) \(=\) \( \frac{s_u}{N}\)
\(RMSE_u\) \(=\) \( \sqrt{\frac{\sum_{i=1}^s |Pr_i-Acr_i| }{s_u}}\)

Keterangan:
\(u\) \(=\) Target user
\(s_u\) \(=\) Jumlah irisan data rekomendasi dan data ground-truth untuk target user \(u\)
\(N\) \(=\) Jumlah rekomendasi
\(I\) \(=\) Data seluruh film
\(l_u\) \(=\) Data rekomendasi film yang sudah terurut secara descending untuk target user \(u\), dimana \(l_u \in I\)
\(Top(l_u,N)\) \(=\) Data rekomendasi film untuk target user \(u\)
\(Test_u\) \(=\) Data ground-truth dari target user \(u\)
\(Pr_u\) \(=\) Nilai prediksi rating dari item \(i\) oleh target user \(u\)
\(Acr_u\) \(=\) Nilai rating yang sebenarnya dari item \(i\) oleh target user \(u\)
